# Gomes Auto Works — Landing Page

Single-page email-capture landing page for **Gomes Auto Works**.
Servicing, repairs, diagnostics & performance — all makes and models, BMW & Mini specialist.

## Files
- `index.html` — the whole site (self-contained, no dependencies)
- `assets/logo.png` — **add your logo here** (the chrome "GOMES AUTO WORKS" badge). Until it's added, a styled text version shows automatically.

## To make the email form work
The form currently posts to a placeholder. Sign up at [formspree.io](https://formspree.io) (free), create a form, then in `index.html` replace:

```
action="https://formspree.io/f/your-form-id"
```

with your real Formspree endpoint. Signups will then arrive in your inbox.

## To fill in
- Phone, email and address in the footer (currently placeholders).

## Hosting (free, via GitHub Pages)
Repo Settings → Pages → deploy from `main` / root. Site will be live at the Pages URL.
